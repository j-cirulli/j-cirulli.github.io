Final Tutorial: Predicting College Quarterback's success in the NFL¶

Every year, struggling nfl franchise turn to the NFL Draft to pick a new quarterback. Often regarded as the most important player on the team, quarterbacks have to use a lot of skills, and it has been an ongoing challenge to determine what a quarterback's potential is, and how their game will translate to the next level.

External Reading:

  • Discussion of NFL Quarterback evaluation challenges: http://www.espn.com/nfl/story/_/id/23039883/history-tells-us-nfl-terrible-evaluating-quarterbacks-means-2018-draft-prospects
  • Top 50 Quarterback Busts of All-Time: https://bleacherreport.com/articles/1047730-50-biggest-quarterback-busts-in-nfl-draft-history#slide49

In this tutorial, I look to examine how some factors regarding collegiate quarterbacks tend to translate to professionals, especially in the capacity of quarterbacks considered for first round draft picks (drafted before or on the 128th pick). The factors examined were:

  1. Height
  2. Weight
  3. Completion Percentage
  4. TD to Interception Ratio
  5. College Quarterback Rating

From these characteristics, multivariable linear regression was used to predict NFL Quarterback Rating, which was deemed as the best way to determine NFL Quarterback success.

II. Get NFL Stats, Height, Weight¶

Remove all rows that do not have all necessary data (no NFL stats or unretrievable), keeping track of lost data numbers

Data is from: https://www.pro-football-reference.com/

  • Iterate through each row of dataframe
  • Search for player by name by using the proper url (format seen below)
  • From Result 1 of 2 things occurs:
    1. If the player is not unique (other players have the same name), taken to page that lists all of the instances of the name
      • Extract player's correct bio by using school information, parsing the information using regular expressions. Redirect to bio
    2. If the player is unique, the page is automatically taken to their bio
  • At bio:
    • Find table holding passing data and extract career passer rating using regular expressions. Add to dataframe
    • Find area that holds height and weight and extract values to put into dataframe

NOTE: This process takes substantial time as it needs to access many webpages

    first = name[0]
    last = name[1]
    
    # lowercase version search url
    firstname = first.lower()
    lastname = last.lower()
    
    # add on Jr, II, etc
    if(len(name) == 3):
        last += " " + name[2]
        lastname = last.lower()

    url = "https://www.pro-football-reference.com/search/search.fcgi?hint=" + firstname + \
    "+" + lastname + "&search=" + firstname + "+" + lastname + "&pid=&idx="
    
    # get url data
    r = requests.get(url,headers=headers)
    
    # use beautiful soup to get the contents of the page
    root = BeautifulSoup(r.content)
    
    # try to find table signifying search-results
    table = root.find('div',class_="search-results")
    
    
    # table was present --> non-unique name, has search results
    if(table != None):
        
        # all exceptions point to lack of necessary data, therefore handle by dropping row
        try:

            table = str(table).replace("\n","")

                    link_data = ele

                regex = r"([0-9]*)cm,\W*([0-9]*)kg"
                data = re.findall(regex,info)[0]
                df.at[index,'height_cm'] = int(data[0])
                df.at[index,'weight_kg'] = int(data[1])
            #invalid data
            else:
                df = df.drop(index)
                lost_data_nfl += 1
        #missing data

    # get name, school data. Only need lower case versions
    name = row['name'].split()
    school = row['college'].lower()
    firstname = name[0].lower()
    lastname = name[1].lower()
    if(len(name) == 3):
        lastname += " " + name[2].lower()
    
    url = "https://www.sports-reference.com/cfb/search/search.fcgi?hint=" + firstname + \
    "+" + lastname + "&search=" + firstname + "+" + lastname + "&pid=&idx="

    root = BeautifulSoup(r.content)
    table = root.find('div',class_="search-results")
    
    # multiple of same name, search results

        try:
            table = str(table).lower().replace("\n","")
            elements = table.split("<a")

            regex = r"href=\"(.*)\">" + firstname + " " + lastname

            table = str(root.find('table').find('tfoot'))

            "[0-9]*)<[\s\S]*?\"pass_int\">([0-9]*)<[\s\S]*?\"pass_rating\">([0-9\.]*)<[\s\S]*?<\/tr>"

I. Extracting Player's Collegiant Data:¶

Data is from: https://www.sports-reference.com/cfb/ (used earlier for previously drafted players as well)

Differences:

  • DataFrame is not yet made, so data is simply added to arrays that will be used during the creation of the dataframe later
  • candidate array is iterated over
  • candidates are held in tuples (NAME, SCHOOL), so name and school information is extracted slightly differently
  • Height and weight are extracted from this site. Although it is the cfb version, the same exact procedure is used as on: https://www.pro-football-reference.com/

## candidates: found from web research and manually inputted
candidates = [("Justin Herbert","Oregon"), ("Dwayne Haskins", "Ohio State"), \
              ("Daniel Jones", "Duke"), ("Will Grier", "West Virginia"), ("Drew Lock", "Missouri"), \
              ("Ryan Finley", "North Carolina State"), ("Kyler Murray", "Oklahoma")]

# arrays to populate for dataframe
names = []
heights = []
weights = []
comp_percs = []
td_ints = []
ratings = []

# iterate through candidates
for qb,school in candidates:
    
    # extract name and school
    name = qb.split()
    firstname = name[0].lower()
    lastname = name[1].lower()
    names.append(name[0] + " " + name[1])
    school = school.lower()
    
    # access search url
    url = "https://www.sports-reference.com/cfb/search/search.fcgi?hint=" + firstname + "+" + lastname + "&search=" + firstname + "+" + lastname + "&pid=&idx="
    r = requests.get(url,headers=headers)
    # use beautiful soup to get the contents of the page
    root = BeautifulSoup(r.content)
    table = root.find('div',class_="search-results")
    
    # non-unique name --> find correct search result
    if(table!= None):
        
        # get correct search result
        table = str(table).lower().replace("\n","")
        elements = table.split("<a")
        link_data = ""
        regex = r"href=\"(.*)\">(<strong>){0,1}" + firstname + " " + lastname
        for ele in elements:
            if(school in ele):
                link_data = ele
                
        # get player bio url info from result
        data = re.findall(regex,link_data)[0]
        url2 = 'https://www.sports-reference.com' + data[0]
        
        # access bio
        r2 = requests.get(url2,headers=headers)
        root = BeautifulSoup(r2.content)
        
        # get data from table (same as before)
        table = str(root.find('table').find('tfoot'))
        regex = r"<tr[\s\S]*?career[\s\S]*?\"pass_cmp_pct\">([0-9\.]*)<[\s\S]*?\"pass_td\">([0-9]*)<[\s\S]*?\"pass_int\">([0-9]*)<[\s\S]*?\"pass_rating\">([0-9\.]*)<[\s\S]*?<\/tr>"
        data = re.findall(regex,str(table).lower())[0]
        rate = data[3]
        td = data[1]
        ints = data[2]
        comps = data[0]   
        
        # append data to arrays rather than using dataframes
        ratings.append(float(rate))
        td_ints.append(float(td) / float(ints))
        comp_percs.append(float(comps))
        
        # append height and weight data in same procedure as before
        info = str(root.find('div', id="info"))
        regex = r"([0-9]*)cm,\W*([0-9]*)kg"
        data = re.findall(regex,info)[0]
        heights.append(int(data[0]))
        weights.append(int(data[1]))
        
      
    # unique name, at player's bio
    else:
        
        # get data from table (same as before)
       
        table = str(root.find('table').find('tfoot'))
        regex = r"<tr[\s\S]*?career[\s\S]*?\"pass_cmp_pct\">([0-9\.]*)<[\s\S]*?\"pass_td\">([0-9]*)<[\s\S]*?\"pass_int\">([0-9]*)<[\s\S]*?\"pass_rating\">([0-9\.]*)<[\s\S]*?<\/tr>"
        data = re.findall(regex,str(table).lower())[0]
